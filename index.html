Google API Configuration

To access your Google Drive video files, you must configure your own Google Cloud Console credentials. This ensures you have your own API rate limits and full control over your data.

How to set up in 2 minutes:

  1. Go to the Google Cloud Console.
  2. Create a new project, enable the Google Drive API.
  3. Go to APIs & Services > Credentials, click Create Credentials > OAuth client ID.
  4. Set Application Type to Desktop app.
  5. Add Redirect URI: http://127.0.0.1:54321/auth/callback
  6. Copy the Client ID and Client Secret, then paste them below and save.
